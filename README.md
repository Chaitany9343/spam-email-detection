# Spam Email Detection

A machine learning project that detects whether an email is spam or not using Natural Language Processing (NLP) and a Naive Bayes classifier.

This project uses text preprocessing, feature extraction (TF-IDF), and a Naive Bayes model to classify messages as spam or ham.
